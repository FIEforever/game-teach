## 📌 项目背景

传统教学中，知识点练习往往以枯燥的刷题为主，学生缺乏学习兴趣。趣学园平台将知识点逐一转化为精美的 HTML5 互动小游戏，涵盖 15+ 种不同玩法，让学生在游戏中自然掌握知识。

平台覆盖**小学一至六年级**的**语文、数学、英语**三大学科，教师还可以通过内置的**创作工坊**，利用 6 种游戏模板和 AI Prompt，为任意知识点快速生成配套游戏。

同时，平台提供**班级管理、学生画像、知识点分析**等教学辅助功能，帮助教师实时了解全班对每个知识点的掌握情况。

## ✨ 核心功能

### 📚 三大学科全覆盖
- 覆盖**语文、数学、英语**三大学科
- 支持**一至六年级**全学段
- 五层数据模型：学科 → 年级 → 学期 → 单元 → 知识点

### 🎯 知识点游戏化（演示：四年级数学）
- 覆盖人教版四年级下册数学 **9 个单元、32 个知识点**
- 每个知识点至少配备 **1 个专属小游戏**
- 共 **32 个 HTML5 精美小游戏**，15+ 种不同玩法

### 🛠️ 教师创作工坊
- **6 种游戏模板**：选择题闯关、拖拽配对、填空挑战、记忆翻牌、数学速算、拼图游戏

- **跨学科适配**：模板支持语文（汉字消消乐）、英语（单词拼写）等不同学科

### 🏫 班级学生管理
- 支持 **6 个年级 × 4 个班级** 的完整学校架构
- 403 班 **45 名真实学生**名单录入
- 搜索过滤、头像卡片、一键选择

## 🛠️ 技术架构

```
┌─────────────────────────────────────────────────┐
│                   前端 (SPA)                      │
│         原生 HTML / CSS / JavaScript              │
│    无框架依赖 · 零构建工具 · 秒级加载            │

│                   API 层                          │
│         Express.js RESTful API                   │
│   /api/school  /api/knowledge  /api/games        │

│                   数据层                          │
│  本地: SQLite (better-sqlite3)                   │
│  Vercel: 内存 JSON 数据库 (自动切换)             │
│  五层结构: 学科→年级→学期→单元→知识点            │
├─────────────────────────────────────────────────┤
│                 HTML5 游戏引擎                     │
│     单文件自包含 · 无外部依赖 · 移动端适配        │
│     6种模板 · 15+种玩法 · 跨学科适配              │
└─────────────────────────────────────────────────┘
```

### 技术选型

| 层级 | 技术 | 理由 |
|-----|------|------|
| 前端 | 原生 HTML/CSS/JS | 零构建、快速迭代、易于理解 |
| 后端 | Node.js + Express | 轻量、Serverless 友好 |
| 数据库 | SQLite / 内存 JSON | 本地开发用 SQLite，Vercel 部署自动切换内存模式 |
| 游戏 | 纯 HTML5 单文件 | 无依赖、即开即玩、易于扩展 |
| 部署 | Vercel | 免费额度、自动 HTTPS、全球 CDN |

## 🚀 快速开始

### 方式一：本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/FIEforever/game-teach.git
cd game-teach

# 2. 安装依赖
cd server && npm install

# 3. 初始化数据库（含全部种子数据）
node init-db.js

# 4. 启动服务
node app.js

# 5. 打开浏览器访问
# http://localhost:3000
```

### 方式二：一键部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/FIEforever/game-teach)

1. 点击上方按钮
2. 使用 GitHub 账号登录 Vercel
3. 确认部署，等待 1-2 分钟
4. 自动获得线上地址

> Vercel 部署使用内存 JSON 数据库，数据在每次冷启动时重置。如需持久化，建议本地运行。

## 📖 使用流程

```
选择班级 → 选择学生 → 选择学科 → 浏览知识点 → 选择游戏 → 游戏化学习 → 查看成绩
    │           │           │           │           │           │           │
    ▼           ▼           ▼           ▼           ▼           ▼           ▼
 24个班级    45名学生    语/数/英     9个单元      32个游戏    8-12关      自动记录
 按年级分组  搜索过滤    三大学科     32个知识点   多种玩法    评分+星级    历史可查
```

### 教师视角
1. 进入平台 → 选择班级（如四年级3班）
2. 点击 **📊 班级分析** → 查看各知识点掌握情况
3. 识别薄弱知识点 → 针对性布置游戏练习
4. 使用 **🛠️ 创作工坊** → 为新知识点生成游戏

### 学生视角
1. 选择自己的名字 → 进入知识点页面
2. 选择想练习的知识点 → 选择游戏
3. 完成游戏 → 查看成绩和星级
4. 点击头像 → 查看个人学习仪表板

## 🔧 扩展指南

### 添加新游戏
1. 在 `server/uploads/games/` 下创建新目录
2. 编写 `index.html`（单文件，内嵌 CSS + JS）
3. 在 `server/init-db.js` 的 `defaultGames` 数组中添加记录
4. 重新运行 `node init-db.js`

### 添加新知识点
1. 在 `server/init-db.js` 的 `units` 数组中添加
2. 关联对应的游戏

### 接入其他年级/学科
1. 修改 `init-db.js` 中的种子数据
2. 按相同格式添加学科、年级、单元和知识点
3. 利用创作工坊模板为每个知识点创建配套游戏
